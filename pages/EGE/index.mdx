## Формат экзамена

С 2021 года ЕГЭ по информатике проводится в компьютерной форме (КЕГЭ). Экзамен состоит из 27 заданий различного уровня сложности, которые выполняются с использованием компьютера.

```markdown
  1. Время на выполнение: 3 часа 55 минут (235 минут)
  2. Максимальный балл: 29 первичных баллов
  3. Минимальный проходной балл: 40 тестовых баллов (после перевода)
  4. Для сдачи необходимо иметь базовые навыки программирования
```

## Структура экзамена

Задания ЕГЭ по информатике затрагивают следующие тематические блоки:

```markdown
1. Информация и информационные процессы
2. Моделирование и компьютерный эксперимент
3. Системы счисления
4. Логика и алгоритмы
5. Элементы теории алгоритмов
6. Программирование
7. Архитектура компьютеров и компьютерных сетей
8. Обработка числовой информации
9. Технологии поиска и хранения информации
```

## Языки программирования

<Callout type="info" title="Инфо">
  Вы можете выбрать любой язык программирования из предложенных для решения задач, но рекомендуется использовать тот, с которым вы лучше всего знакомы.
</Callout>

Наиболее популярные языки среди сдающих:

```python
# Пример решения задачи на Python
def find_divisors(n):
    count = 0
    for i in range(1, n + 1):
        if n % i == 0:
            count += 1
    return count

n = int(input())
print(find_divisors(n))
```
