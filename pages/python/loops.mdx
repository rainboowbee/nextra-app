# Циклы в Python: полное руководство

Циклы — одна из фундаментальных концепций программирования, позволяющая многократно выполнять блок кода. В Python существует несколько типов циклов, каждый с собственными уникальными особенностями и применением.

<Callout type="info" title="Важно знать">
  Циклы позволяют автоматизировать повторяющиеся задачи и значительно сокращают
  объем кода, который вам нужно написать.
</Callout>

## Типы циклов в Python

Python предлагает два основных типа циклов:

- Цикл `for` — используется для итерации по последовательностям
- Цикл `while` — выполняется, пока условие остается истинным

## Цикл for

Цикл `for` в Python отличается от аналогичного цикла в других языках программирования. Он работает скорее как итератор, перебирая элементы последовательностей (списки, кортежи, словари, строки и др.).

```python
fruits = ["яблоко", "банан", "вишня"]
for fruit in fruits:
    print(fruit)
```

<Callout type="success" title="Результат">
  яблоко банан вишня
</Callout>

### Использование range()

Функция `range()` часто используется с циклом `for` для выполнения кода определенное количество раз:

```python
for i in range(5):
    print(i)  # Выведет числа от 0 до 4
```

<Callout type="info">
  Функция range() создает последовательность чисел. По умолчанию она начинается
  с 0, увеличивается на 1 и заканчивается перед указанным числом.
</Callout>

## Цикл while

Цикл `while` выполняет блок кода, пока указанное условие остается истинным.

```python
count = 0
while count < 5:
    print(count)
    count += 1  # Не забывайте увеличивать счетчик
```

<Callout type="warning" title="Предупреждение">
  Если условие в цикле `while` никогда не станет ложным, цикл будет выполняться
  бесконечно. Всегда обеспечивайте способ выхода из цикла!
</Callout>

## Управляющие операторы

<Steps>
  <Steps.Step title="Оператор break">
    Оператор `break` позволяет досрочно выйти из цикла: ```python for i in
    range(10): if i == 5: break print(i) # Выведет числа от 0 до 4 ```
  </Steps.Step>

  <Steps.Step title="Оператор continue">
    Оператор `continue` пропускает текущую итерацию цикла и переходит к
    следующей: ```python for i in range(7): if i == 3 or i == 6: continue
    print(i) # Выведет 0, 1, 2, 4, 5 ```
  </Steps.Step>

  <Steps.Step title="Оператор else в циклах">
    Блок `else` в циклах Python выполняется, когда цикл завершается нормально
    (без использования `break`): ```python for i in range(5): print(i) else:
    print("Цикл завершен нормально") ```
  </Steps.Step>
</Steps>

## Вложенные циклы

Python позволяет создавать вложенные циклы – циклы внутри других циклов:

```python
for i in range(3):
    for j in range(3):
        print(f"({i}, {j})")
```

<Callout type="error" title="Внимание к производительности">
  Вложенные циклы выполняются медленнее, так как внутренний цикл запускается
  полностью для каждой итерации внешнего цикла. Будьте осторожны с большими
  наборами данных!
</Callout>

## Генераторы списков

Генераторы списков предлагают элегантный способ создания списков на основе существующих последовательностей:

```python
# Обычный способ
squares = []
for x in range(10):
    squares.append(x**2)

# С помощью генератора списков
squares = [x**2 for x in range(10)]
```

<Callout type="success" title="Совет">
  Генераторы списков обычно более компактны и работают быстрее, чем
  эквивалентные циклы for с операторами append.
</Callout>

## Практические примеры использования циклов

### Подсчет суммы чисел

```python
numbers = [1, 2, 3, 4, 5]
total = 0

for num in numbers:
    total += num

print(f"Сумма: {total}")  # Выведет: Сумма: 15
```

### Поиск элемента в списке

```python
items = ["книга", "ручка", "карандаш", "тетрадь"]
search_for = "ручка"

for item in items:
    if item == search_for:
        print(f"Найдено: {item}")
        break
else:
    print("Элемент не найден")
```
