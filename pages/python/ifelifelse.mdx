# Условные конструкции в Python

Условные конструкции позволяют управлять выполнением кода в зависимости от заданных условий.

## Основные операторы

Python использует следующие ключевые слова для создания условий:

- `if` — выполняет блок кода, если условие истинно.
- `elif` — позволяет проверить дополнительные условия.
- `else` — выполняет блок кода, если все предыдущие условия ложны.

## Примеры использования

<Steps>
  <Steps.Step title="Простое условие">

```python
age = 18
if age >= 18:
    print("Вы совершеннолетний")
```

Выведет: `Вы совершеннолетний`

  </Steps.Step>

  <Steps.Step title="Использование elif">

```python
score = 75
if score >= 90:
    print("Отлично!")
elif score >= 70:
    print("Хорошо!")
else:
    print("Попробуйте еще раз")
```

Выведет: `Хорошо!`

  </Steps.Step>

  <Steps.Step title="Использование else">

```python
number = -5
if number > 0:
    print("Число положительное")
else:
    print("Число отрицательное или ноль")
```

Выведет: `Число отрицательное или ноль`

  </Steps.Step>

  <Steps.Step title="Тернарный оператор">

```python
x = 10
y = 20
max_value = x if x > y else y
print(f"Максимальное значение: {max_value}")
```

Выведет: `Максимальное значение: 20`

  </Steps.Step>

  <Steps.Step title="Вложенные условия">

```python
num = 10
if num > 0:
    if num % 2 == 0:
        print("Положительное четное число")
    else:
        print("Положительное нечетное число")
else:
    print("Число отрицательное или ноль")
```

Выведет: `Положительное четное число`

  </Steps.Step>
</Steps>

<Callout type="warning" title="Важно">
  Избегайте слишком глубоко вложенных условий — это делает код сложнее для
  понимания.
</Callout>
