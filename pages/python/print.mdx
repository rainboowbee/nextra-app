```python
x = 5
y = 10
print(x, y)
x, y = y, x
print(x, y)
```

Что будет выведено на экран в результате выполнения этой программы?

Ответ:

```
5 10
10 5
```

## Полезные советы

1. Используйте f-строки для более читаемого форматирования.
2. Помните, что `print()` автоматически добавляет перевод строки в конце.
3. Для отладки программ удобно выводить промежуточные значения переменных.
4. В заданиях ЕГЭ внимательно следите за типами данных при выводе.
