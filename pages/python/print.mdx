# Функция `print` в Python

Функция `print` используется для вывода информации в консоль. Это одна из самых часто используемых функций в Python.

## Синтаксис

```python
print(*objects, sep=' ', end='\n')
```

### Основные параметры:

- `*objects` — один или несколько объектов, которые нужно вывести.
- `sep` — строка-разделитель между объектами (по умолчанию пробел `' '`).
- `end` — строка, которая добавляется в конце вывода (по умолчанию `\n`).

<Callout type="info" title="Важно">
  `print` может принимать любое количество аргументов и автоматически
  преобразует их в строковый формат.
</Callout>

## Примеры использования

```python
print("Привет, мир!")
```

Выведет: `Привет, мир!`

```python
print("Python", "лучший", "язык", sep=" - ")
```

Выведет: `Python - лучший - язык`

```python
print("Это первая строка", end=" ")
print("и она продолжается.")
```

Выведет: `Это первая строка и она продолжается.`

## f строка

В функции print можно использовать форматированную строку

```python
name = "Алиса"
age = 25
print(f"Меня зовут {name}, мне {age} лет.")
```

Выведет: `Меня зовут Алиса, мне 25 лет.`

Для того чтобы вывести значение из переменной в строке, необходимо использовать фигурные скобки {}, а внутри написать название переменной, данные которой мы хотим "вшить" в строку
