# Дополнительные модули в Python

В этом разделе рассматриваются стандартные модули Python, которые помогают работать с операционной системой, временем, системными параметрами и многим другим.

## Что такое модули?

Модули в Python — это файлы с кодом, содержащие функции, классы и переменные, которые можно переиспользовать в разных программах. Python поставляется с множеством встроенных модулей, которые можно импортировать и использовать без установки.

## Популярные встроенные модули

В данном разделе мы рассмотрим несколько полезных модулей:

- **ipaddress** — работа с ip адресами
- **itertools** — перебор различных комбинаций
- **turtle** — работа черепашкой

Перейдите к соответствующим разделам, чтобы узнать подробнее о каждом из них.

## Как импортировать модули?

Чтобы использовать модуль, его нужно импортировать с помощью команды `import`:

```python
import itertools
```

либо можете указать конкретный модуль, который хотите импортировать

```python
from itertoosl import product
```
