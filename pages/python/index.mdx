# Введение в Python

Python — высокоуровневый, интерпретируемый язык программирования общего назначения. Его простой синтаксис и читаемость делают его идеальным языком для изучения программирования, в том числе для подготовки к ЕГЭ и ОГЭ по информатике.

## Почему Python?

- **Простой и понятный синтаксис** — идеален для начинающих
- **Широкое применение** — от веб-разработки до анализа данных и машинного обучения
- **Популярность в заданиях ЕГЭ** — часто используется для решения алгоритмических задач
- **Богатая стандартная библиотека** — множество готовых инструментов

## Установка Python

Для начала работы с Python необходимо установить интерпретатор. Рекомендуется использовать Python 3, так как Python 2 больше не поддерживается.

1. Скачайте последнюю версию Python с [официального сайта](https://www.python.org/downloads/)
2. Запустите установщик и следуйте инструкциям
3. Обязательно отметьте опцию "Add Python to PATH" при установке

## Интегрированные среды разработки (IDE)

Для удобства работы с Python рекомендуется использовать интегрированные среды разработки:

- **PyCharm** — мощная среда разработки с множеством функций
- **Visual Studio Code** — легковесный редактор с поддержкой Python через расширения
- **IDLE** — простая среда, входящая в стандартную установку Python
- **Thonny** — IDE для начинающих, хорошо подходит для обучения
