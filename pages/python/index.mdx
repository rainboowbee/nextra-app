# Начало работы с Python и VSCode

<Callout type="info" title="Важно">
  Прежде чем начать разработку, необходимо правильно настроить вашу среду. Это
  руководство поможет установить все необходимые инструменты.
</Callout>

## Установка Python

<Steps>
  <Steps.Step title="Скачайте установщик Python">
    Перейдите на [официальный сайт Python](https://www.python.org/downloads/) и скачайте последнюю стабильную версию для вашей операционной системы.
  </Steps.Step>
  
  <Steps.Step title="Запустите установщик">
    **Для Windows:**
    - Обязательно отметьте опцию **"Add Python to PATH"**
    - Выберите **"Install Now"** для стандартной установки

    **Для macOS:**
    - Запустите скачанный `.pkg` файл
    - Следуйте инструкциям установщика

    **Для Linux:**
    ```bash
    # Ubuntu/Debian
    sudo apt update
    sudo apt install python3 python3-pip

    # Fedora
    sudo dnf install python3 python3-pip

    # Arch Linux
    sudo pacman -S python python-pip
    ```

  </Steps.Step>
  
  <Steps.Step title="Проверьте установку">
    Откройте терминал (командную строку) и введите:

    ```bash
    # Windows
    python --version

    # macOS и Linux
    python3 --version
    ```

    Вы должны увидеть номер установленной версии Python.

  </Steps.Step>
</Steps>

## Установка и настройка VSCode

<Steps>
  <Steps.Step title="Установите VSCode">
    1. Перейдите на [сайт Visual Studio Code](https://code.visualstudio.com/)
    2. Скачайте и установите версию для вашей ОС
  </Steps.Step>
  
  <Steps.Step title="Установите расширения для Python">
    Откройте VSCode и перейдите во вкладку Extensions (Расширения). Установите следующие расширения:

    - **Python** от Microsoft
    - **Pylance** для улучшенной поддержки Python
    - **Python Indent** для правильных отступов

    ![Расширения VSCode для Python](/images/vscode-python-extensions.png)

  </Steps.Step>
  
  <Steps.Step title="Настройте интерпретатор Python">
    1. Откройте командную палитру: `Ctrl+Shift+P` (Windows/Linux) или `Cmd+Shift+P` (macOS)
    2. Введите **Python: Select Interpreter** и выберите установленную версию Python
  </Steps.Step>
  
  <Steps.Step title="Создайте первый проект">
    1. Создайте новую папку для вашего проекта
    2. Откройте эту папку в VSCode
    3. Создайте новый файл с расширением `.py`
    4. Напишите тестовый код:

    ```python
    print("Hello, Python!")
    ```

    5. Запустите код, нажав кнопку ▶️ в правом верхнем углу

  </Steps.Step>
</Steps>

## Настройка виртуального окружения

<Callout type="warning" title="Лучшие практики">
  Рекомендуется использовать виртуальные окружения для каждого проекта, чтобы
  избежать конфликтов между зависимостями.
</Callout>

```bash
# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
# Windows
venv\Scripts\activate

# macOS и Linux
source venv/bin/activate
```

После активации виртуального окружения установите необходимые пакеты:

```bash
pip install package-name
```

## Полезные настройки VSCode

<Callout type="success" title="Совет">
  Следующие настройки значительно улучшат ваш опыт разработки на Python в
  VSCode.
</Callout>

Для более комфортной работы с Python в VSCode рекомендуется добавить следующие настройки в ваш `settings.json`:

```json
{
  "python.linting.enabled": true,
  "python.linting.pylintEnabled": true,
  "python.formatting.provider": "black",
  "editor.formatOnSave": true,
  "python.analysis.typeCheckingMode": "basic"
}
```

Эти настройки включают линтер и автоматическое форматирование кода.
